view = the entire file

## process

display mode

- display view through window
- inputs:
        -       move window
        - (.)   enter command mode

- allow user to enter commands that do not modify a view
- show command being entered along with suggestions
- inputs:
        - edit command
        - select and choose command
- commands:
        - see `file`
        - end
